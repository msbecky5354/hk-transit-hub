User Manual

貼地通 (HK Transit Hub) 使用指南

設計初衷 (Objective)

貼地通 (HK Transit Hub) 旨在為香港市民打造一個「極速、零廣告、高度個人化」的日常通勤儀表板。

有別於傳統的交通 App 需要每次重新輸入路線和車站，本程式讓你將最常用的路線（如「屋企落樓」、「放工轉車」）一次性釘選在首頁。打開網頁，所有目標車次的時間立刻一目了然，助你精準掌握出門時機。

頂部列功能 (Header Features)

分享應用程式 (Share App)

點擊頂部的分享圖示，你可以快速將「貼地通」的網址發送給親朋好友。

安裝至手機 (Install PWA)

iOS (Safari): 確保使用 Safari 瀏覽器開啟網站 -> 點擊底部「分享」圖示（正方形加向上箭嘴） -> 點選「加到主畫面」 (Add to Home Screen) -> 點擊右上角「新增」即可。
Android (Chrome): 使用 Chrome 瀏覽器開啟網站 -> 點擊右上角「三個點」選單 -> 點選「安裝應用程式」 (Install app) 或「加到主畫面」 -> 根據提示完成安裝。

安装至手机 (Install PWA)

iOS (Safari): 必须使用 Safari 浏览器打开网站 -> 点击底部“分享”图标（正方形加向上箭头） -> 向下滑动点选“添加到主屏幕” (Add to Home Screen) -> 点击右上角“添加”即可完成。
Android (Chrome): 使用 Chrome 浏览器打开网站 -> 点击右上角“三个点”菜单 -> 点选“安装应用” (Install app) 或“添加到主屏幕” -> 根据提示完成安装。

Install to Home Screen (PWA)

iOS (Safari): Open this site in Safari -> Tap the "Share" icon (square with an up arrow) at the bottom toolbar -> Select "Add to Home Screen" -> Tap "Add" in the top right corner.
Android (Chrome): Open this site in Chrome -> Tap the "Three Dots" menu icon in the top right -> Select "Install app" or "Add to Home Screen" -> Follow the prompts to complete.

切換語言 (Language Switch)

點擊 繁 | 简 | EN 即可即時切換整個應用程式的顯示語言。

基礎操作 (Basic Usage)

1

新增路線 (Add Routes)

點擊底部 新增 進入設定中心，選擇交通工具、搜尋路線，並將其釘選到你的儀表板。

2

排序與整理 (Drag & Reorder)

在儀表板中，長按群組標題旁邊的 或卡片右側的 ，即可上下拖曳，將最常用的路線排在最頂。

3

編輯與刪除 (Edit & Delete)

將路線卡片向左滑動 (Swipe Left)，即可喚出隱藏選單，點擊 可為路線自訂專屬花名，點擊紅色垃圾桶即可刪除。你亦可點擊群組標題旁的 一鍵刪除整個群組。如需修改自訂群組名稱，請前往「新增」頁面，選擇該群組後點擊旁邊的 編輯按鈕。

4

數據備份 (Backup Data)

點擊底部 設定，你可以隨時將目前的路線佈局匯出成 JSON 檔案。日後更換手機或清理瀏覽器數據後，只需匯入該檔案即可瞬間復原。
